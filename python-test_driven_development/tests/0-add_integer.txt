===========================
Cómo utilizar 0-add_integer.py
===========================

Descripción
===========

Este módulo define una función de suma de enteros ``add_integer(a, b=98)``.

Uso
=====

``add_integer(...)``` devuelve la suma de sus dos argumentos. Para números,
ese valor es equivalente a usar el operador ``+``.

::

    >>> add_integer = __import__('0-add_integer').add_integer
    >>> add_integer(2, 3)
    5

::

    >>> add_integer(2, -3)
    -1

La función también funciona con valores de punto flotante.

::

    >>> add_integer(2.0, 3.0)
    5

Tenga en cuenta que los flotantes se convierten a enteros antes de que se realice la adición.

::

    >>> add_integer(2.9, 0.2)
    2

::

    >>> add_integer(-2.9, -0.2)
    -2

Los valores de punto flotante y no de punto flotante se pueden combinar.

::

    >>> add_integer(2.3, -3)
    -1

El segundo argumento es opcional; de forma predeterminada, es 98.

::

    >>> add_integer(2)
    100

No números
===========

``add_integer()`` espera que ambos argumentos sean enteros o flotantes. Si alguno de los argumentos es un no entero y no flotante, se genera un TypeError:

::

    >>> add_integer("hello", 3)
    Traceback (most recent call last):
    TypeError: a must be an integer

::

    >>> add_integer(2, "hello")
    Traceback (most recent call last):
    TypeError: b must be an integer

::

    >>> add_integer(None)
    Traceback (most recent call last):
    TypeError: a must be an integer

::

    >>> add_integer(2.3, None)
    Traceback (most recent call last):
    TypeError: b must be an integer

Si ambos argumentos no son enteros y no flotantes, se imprime un mensaje de TypeError solo para el primer argumento.

::

    >>> add_integer("hello", "there")
    Traceback (most recent call last):
    TypeError: a must be an integer

La función fallará si se proporciona infinito.

::

    >>> add_integer(float('inf'))
    Traceback (most recent call last):
    OverflowError: cannot convert float infinity to integer

::

    >>> add_integer(2, float('inf'))
    Traceback (most recent call last):
    OverflowError: cannot convert float infinity to integer

Y nuevamente con números NaN.

::

    >>> add_integer(float('nan'))
    Traceback (most recent call last):
    ValueError: cannot convert float NaN to integer

::

    >>> add_integer(2, float('nan'))
    Traceback (most recent call last):
    ValueError: cannot convert float NaN to integer
